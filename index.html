Face Recognition

A real-time face detection & identification system powered by deep learning encodings. Recognizes known faces from a live webcam stream and displays names & ages instantly.

See How It Works View Code
Features

What It Can Do

Built with cutting-edge libraries to deliver real-time, accurate face identification.

Real-Time Detection

Continuously scans each webcam frame to detect and locate all visible faces in milliseconds.

Identity Recognition

Matches detected faces against a library of known people using 128-dimensional deep learning encodings.

Age Display

Displays the stored age alongside the name for every recognized individual in the frame.

Unknown Handling

Gracefully labels unrecognized faces as "Unknown" — no false identification, no crashes.

Live Bounding Boxes

Draws precise bounding rectangles around each detected face with overlaid name and age text.

Easy Extensibility

Add new people by simply dropping their image into the folder and adding their info to the dictionary.

Pipeline

How It Works

A clean 9-step pipeline from loading known faces to real-time identification.

01

Load Known Faces

Images from the images/ folder are loaded. Each filename becomes a person's name.

02

Generate Encodings

The face_recognition library converts each face into a unique 128-number vector.

03

Open Webcam

OpenCV captures a live video stream from the default camera device.

04

Capture & Convert Frames

Each frame is grabbed and converted from BGR to RGB format for the recognition library.

05

Detect Face Locations

The HOG-based face detector finds the bounding boxes of all faces in the frame.

06

Encode Detected Faces

Every detected face is encoded into a 128-D vector for comparison.

07

Compare & Match

The live encodings are compared to the known encodings to find the best match.

08

Draw Annotations

A green rectangle and a label with the person's name and age are drawn on the frame.

09

Display & Loop

The annotated frame is shown via cv2.imshow(). Press Q to quit cleanly.

Core Code

The main recognition loop — elegant and compact.

code_1.py 
# Load and encode known faces
for file in os.listdir("images"):
    img = face_recognition.load_image_file(f"images/{file}")
    encoding = face_recognition.face_encodings(img)[0]
    known_encodings.append(encoding)
    known_names.append(os.path.splitext(file)[0])

# Age dictionary
ages = {"Amrit": 19, "Jay shri": 50, "Arya": 30}

# Main recognition loop
while True:
    ret, frame = cap.read()
    rgb_frame = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)

    locations = face_recognition.face_locations(rgb_frame)
    encodings = face_recognition.face_encodings(rgb_frame, locations)

    for (top, right, bottom, left), face_enc in zip(locations, encodings):
        matches = face_recognition.compare_faces(known_encodings, face_enc)
        name = "Unknown"
        if True in matches:
            index = matches.index(True)
            name = known_names[index]
        age_text = f", Age: {ages.get(name, 'N/A')}"

        cv2.rectangle(frame, (left, top), (right, bottom), (0, 255, 0), 2)
        cv2.putText(frame, f"{name}{age_text}", (left, top - 10),
                    cv2.FONT_HERSHEY_SIMPLEX, 0.9, (255, 255, 255), 2)

    cv2.imshow("Face Recognition", frame)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break
